Myocontroller

Component Type: Sensor (Subcategory: Bioelectric Measurement)

This component provides an interface to the Myocontroller (OttoBock), which is used to transmit EMG signals to the computer via Bluetooth. Data can be either provided by a COM Port or from a recorded logfile.

Output Port Description

  • Channel1 to Channel6 [integer]: Each output corresponds to the sampled data from its corresponding channel. The values range for 0 to 65535 (16 bit integer).

Properties

  • COMPort [string]: The name of the COM port, for example COM2 or COM17.
  • Baudrate [integer]: The baud rate for the transmission. Standard is 57600.
  • useFile [boolean]: If false (default) the COM port is used for communication, if true the file defined by property 'filename' is opened and provides the data.
  • filename [string]: The filename/path of the file to open, if 'useFile' is true. The path is relative to the bin/ARE/data folder
